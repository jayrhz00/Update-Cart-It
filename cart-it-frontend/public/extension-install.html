Cart-It Extension

Save from any site, share lists and chat with friends, track prices and stock, and add notes. Install in one click from your browser store when available. The Cart-It icon appears next to the address bar.

Manual install (ZIP fallback)

Use this if the store version is still in review or you want today’s newest build before it propagates. The ZIP above is always the latest code.

  1. Click the Download ZIP button for your browser above.
  2. Right-click the downloaded file and choose Extract All… to unzip it into a folder.
  3. Open chrome://extensions (Chrome) or edge://extensions (Edge) in a new tab.
  4. Toggle on Developer mode in the top-right corner.
  5. Click Load unpacked and select the unzipped folder.
  6. The Cart-It icon will appear next to the address bar — pin it for easy access.

Firefox: open about:debugging#/runtime/this-firefox → Load Temporary Add-on… → pick the manifest.json inside the unzipped folder. Temporary add-ons disappear when Firefox restarts; the official store version (when approved) installs permanently.
